# 정밀도 관측은 옵트인 텔레메트리 포트로(미룸)

핵심 성공 지표(precision@1, MRR, 회수 성공률)는 데이터가 로컬·비공개라 출시 후 직접 관측할 수 없다. 골든셋은 개발용일 뿐이다. v1에서는 `TelemetryPort`로 자리만 열어두고 구현은 미룬다. 켤 때는 반드시 옵트인이며, 내용 없이 익명 신호("1등 결과를 클릭했나" 같은)만 보낸다.

## Consequences

- 프라이버시 제품이라 기본은 텔레메트리 OFF다. ADR 0001의 약속과 충돌하지 않도록, 보내더라도 원문·쿼리 내용이 아니라 익명 집계 신호만 보낸다.
- v1은 포트만 두고 어댑터는 만들지 않는다(OUT). 그전까지 품질 판단은 골든셋(ADR 0004)에 의존한다.
