## 자세한 설명

```
100% 기기 안에서. 아무것도 컴퓨터 밖으로 나가지 않습니다.

하루에도 수십 개의 페이지를 읽지만, 정작 그 유용했던 한 가지가 어디 있었는지는 잊어버리죠. Recall이 그걸 해결합니다 - 그것도 완전히 비공개로요.

웹 페이지를 실제로 읽으면, Recall이 그 본문을 깨끗하게 정리해 브라우저 안의 비공개 데이터베이스에 조용히 저장합니다. 나중엔 친구에게 설명하듯, 내 말로, 정확한 키워드가 아니라 '의미'로 검색하면 됩니다. "잠이 안 오는 이유에 대한 그 글"이라고 치면, 지난주에 읽었던 코르티솔과 멜라토닌 이야기를 Recall이 찾아줍니다.

한국어와 영어를 모두 지원합니다. 어느 언어로 검색하든, 어느 언어로 읽었든 찾아냅니다.

진짜로 비공개입니다 - 약속이 아니라, 코드가 그렇게 만들어졌습니다:
- 모든 것이 내 기기에서 돕니다. 검색 색인도, AI 모델도 확장 프로그램 안에 함께 들어 있습니다. 약 107 MB의 IBM Granite 모델이 패키지에 포함돼 있어, 실행 중에 인터넷에서 내려받지 않습니다.
- 네트워크로 나가는 데이터가 0입니다. 확장 프로그램의 보안 정책이 원격 서버 접속 자체를 막습니다. 데이터를 보낼 백엔드 서버가 아예 없습니다.
- 수집·분석 기능 없음, 계정 없음, 광고 없음. 아무것도 모으지 않고 아무것도 팔지 않습니다.

저장할 것을 가립니다 - 그래서 저장하면 안 되는 건 기억하지 않습니다:
- 실제로 읽은 페이지만 저장합니다(머문 시간이 쌓이고, 스크롤했거나 텍스트를 선택한 뒤에요).
- 민감한 사이트는 기본으로 건너뜁니다: 은행, 결제, 로그인, 계정·비밀번호 페이지, 웹메일, 건강 포털, 비밀번호 관리자.
- 검색 결과 페이지, 내부·사설 네트워크 페이지, 아주 짧은 페이지도 건너뜁니다.
- URL을 저장하기 전에 추적용 파라미터(utm_, gclid, fbclid...)를 떼어냅니다.

통제권은 나에게 있습니다:
- 언제든 저장을 일시정지.
- "이 사이트는 기억하지 않기"로 특정 사이트를 영구 차단.
- "이 사이트 기록 지우기"로 한 사이트와 그 하위 도메인을 한 번에 삭제.
- 지금 보는 페이지를 직접 저장하거나 건너뛰기.
- 확장 프로그램을 삭제하면 로컬 데이터베이스도 함께 지워집니다.

Recall은 연구자, 학생, 작가, 그리고 온라인에서 많이 읽는 모든 사람을 위한 도구입니다. 읽은 것을 클라우드에 넘기지 않고도 나만의 비공개 검색 기억으로 갖고 싶은 분들을 위한 것이죠.
```

## 단일 목적 설명 (CWS 필수)

```
Recall의 목적은 하나입니다. 이미 읽은 웹 페이지를 다시 찾도록, 그 본문을 기기에 로컬로 저장하고 의미로 검색하게 해주는 것입니다. 저장과 검색은 모두 기기 안에서 일어나며, 확장 프로그램은 어떤 데이터도 전송하지 않습니다.
```
